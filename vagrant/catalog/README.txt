How to Run
Please ensure you have Python, Vagrant and VirtualBox installed.

$ cd vagrant
$ vagrant up
$ vagrant ssh
Within the virtual machine change in to the shared directory by running

$ cd /vagrant/
$ cd /vagrant/catalog
$ python project.py
